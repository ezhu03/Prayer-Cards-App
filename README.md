# Prayer Cards App

A local-first prayer card web app for names, requests, priorities, prayer history, timed prayer, and offline verse suggestions.

This public repository intentionally includes only the static app files needed for GitHub Pages. Personal prayer cards are not stored in this repository; each user's data stays in their own browser local storage unless they manually export it.

## Use

Open `index.html` locally or publish this repository with GitHub Pages from the `main` branch.
